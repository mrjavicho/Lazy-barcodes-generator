Lazy barcodes generator

A simple display of barcodes using bwipjs-api Barcod API

  • For now, the only supported barcode type is code128.
  • Changing it is pretty simple, just edit this file and use one of the supported barcodes instead.
  • Yo can use , ; and break lines to separate each barcode value
  • Theres no limit for the number of codes, but each row shows 4 barcodes at most
